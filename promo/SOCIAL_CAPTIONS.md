## LinkedIn

I built AiAutomationBot — an AI-powered customer support chatbot designed specifically for small e-commerce stores.

Most chatbot solutions require a database, a cloud account, and three integrations before you see a single response. This one needs a config file and a terminal. You set your products, prices, policies, and tone in `business.json`. The bot handles customer questions, captures leads, and logs everything to an admin dashboard — no SQL, no vendor accounts, no ops overhead.

It runs on Node.js/Express with OpenAI's GPT-4.1-mini and falls back to demo mode without an API key, so evaluation costs nothing. If you're a developer building for e-commerce clients or a store owner who's tired of paying $200/month for a chatbot that doesn't know your own product names — this is worth a look.
